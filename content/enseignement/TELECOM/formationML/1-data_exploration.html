Introduction à l'exploration intéractive de données avec Python et Pandas¶

Author: Alexandre Gramfort, Joseph Salmon

Objectif: Comprendre le contenu des donnes, la structure, les problmes (donnes manquantes / aberrantes)

Les données:¶

MovieLens 1M Data Set contient les notes attribues des films par des utilisateurs du site Movielens. Les donnes sont fournies, mais peuvent tre trouves, si besoin, l'adresse: http://grouplens.org/datasets/movielens/
